# SimpleEchoServer
A simple multi-client JAVA client-server architecture implemented using UDP. 

Note: Compile all the files.

## Fire up the server -
java qServerUdp

## Fire up the clients one by one -

Client 1:
java qClientUdp -port 4447 -server 127.0.0.1

Client 2:
java qClientUdp -port 4447 -server 127.0.0.1

Client 3:
java qClientUdp -port 4447 -server 127.0.0.1

Client 4:
java qClientUdp -port 4447 -server 127.0.0.1
